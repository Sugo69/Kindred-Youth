  • Every scripture-based round shows the source verse live on screen
  • Opening scripture + testimony invitation bookends every session

Three steps to a classroom-ready game.

Set up once. Every Sunday, it takes about a minute.

01

SIGN IN

Sign in with Google. Start a classroom or join with an invite link from a co-teacher. Once approved, your classroom is yours.

02

ADD A LESSON

Paste this week's Come Follow Me URL (or click ⭐ Fill This Week). Kindred pulls the verses and generates both games.

03

HIT PLAY

Cast the Monitor view to the classroom TV. Keep the admin panel on your phone. Tap to reveal, tap to move on.

Content you can put on the classroom TV with confidence.

Every game runs through two review layers before it reaches a teen. The output ships with a compliance report you can audit.

STRUCTURAL RULES

A keyword filter (substances, sexual content, self-harm, violence) runs on every generated string before the game can render.

AI SAFETY REVIEW

A second pass tags each item pass / rewrite / block — using Handbook §13, §37.8, and For the Strength of Youth as its rubric.

POLICY-CITED OUTPUT

Every generated lesson ships with a compliance report naming the policies it was checked against — not a black box.

NO STUDENT PII

Players use anonymous auth. Only teachers sign in. Team names, not student names, appear on the classroom display.
